# asutp-ai-lab
Личная лаборатория инженера АСУ ТП для экспериментов с web и ИИ.

## Цели
- Автоматизация рутинных инженерных задач.
- Генерация идей для приложений и сервисов.
- Шаблоны и инструменты для EPLAN, ПЛК и сопутствующих процессов.

## Структура
- `docs/` — идеи, сценарии автоматизации.
- `web/` — web-проекты и прототипы.
- `scripts/` — скрипты для автоматизации (EPLAN, ПЛК, логи).
- `engineering/` — инженерные шаблоны и примеры.

> В репозитории нет конфиденциальных данных заказчиков.
