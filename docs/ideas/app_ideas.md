# Идеи приложений и инструментов

## 1. Web-панель для инженера АСУ ТП
- Быстрый расчёт токов, кабелей, автоматов.
- Черновая генерация спецификации.

## 2. Генератор шаблонов EPLAN
- Ввод типового шкафа → вывод структуры проекта / макросов.

## 3. Анализ логов ПЛК
- Загрузка логов → поиск аномалий и типовых ошибок.

## 4. Личный кабинет инженера
- Заметки, шаблоны, конструктор ТЗ, типовые решения.

## 5. Инструмент для предварительного подбора оборудования
- Ввод параметров → список типового оборудования (пока просто текст/таблица).
